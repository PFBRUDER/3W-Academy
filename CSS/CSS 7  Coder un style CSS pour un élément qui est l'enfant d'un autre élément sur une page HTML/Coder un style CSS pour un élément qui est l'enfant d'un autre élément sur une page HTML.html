Introduction : La spécificité en CSS

Synopsis : Quand vous fixez la propriété d'un sélecteur comme "p" à une certaine valeur, cette valeur s'applique à tous les "p". Cependant, si vous changez cette même propriété en choisissant une autre valeur pour une occurrence spécifique de p, ce changement va prendre le dessus sur la propriété par défaut.

  • Si vous écrivez p {font-family:Garamond;}, tous les "p" seront en police Garamond.

  • MAIS si vous écrivez li p {font-family:Verdana;}, les "p" à l'extérieur de "li" seront en Garamond et les "p" l'intérieur de "li" seront en Verdana.

  • En résumé : plus vos sélecteurs sont précis (spécifiques), plus votre CSS met des priorités dans l'application des styles.
